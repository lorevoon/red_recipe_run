## Description
Play as Little Red Riding Hood and explore a forest to search for ingredients for grandma's recipes while wolves are sneaking around!

Dome Keeper by Bippinbits was referenced while designing the game.



## Controls
- Arrow keys
  - Movement
- Z key
  - Pick up items during the day
  - Turn on/off the lantern during night
X key
- Drop items during the day
